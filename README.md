# Polynomial Regression (Position Salaries)

This project demonstrates **Polynomial Regression** using the **Position_Salaries.csv** dataset.  
The goal is to predict **Salary** based on the employee **Level**, and compare Linear Regression vs Polynomial Regression for better curve fitting.
